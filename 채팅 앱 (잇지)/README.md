# 채팅 앱 (잇지)
+ React-Native 프레임 워크를 이용하여 구현한 채팅 앱
+ 관리자가 비콘을 두고, 반경을 설정해서 비콘 반경내에 있는 사용자들끼리 채팅을 함
+ Firebase를 이용하였고 관리자 페이지를 따로 만들어 반경 설정과 채팅방 관리를 함
+ <mark>**_소스코드는 회사 보안상의 이유로 정확한 코드로 명시하지 않았음_**</mark>
