# 채팅 앱 (잇지)
+ React-Native 프레임 워크를 이용하여 구현한 채팅 앱
+ 관리자가 비콘을 두고, 반경을 설정해서 비콘 반경내에 있는 사용자와 채팅
+ Firebase를 이용하였고 관리자 페이지를 따로 만들어 관리
